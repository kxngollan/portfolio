# Portfolio Project

This portfolio project consists of three main components: a client interface for public viewing, an admin interface for management, and a server that handles all backend processes.

## Client

The client side of the portfolio is built with ReactJS, utilizing Vite for an optimized build setup. This part of the application is designed for people to view the portfolio content.

### Features:

- **React Icons and Devicons**: Utilized for adding scalable and customizable icons.
- **React Router DOM**: Manages the routing of the application, allowing navigation between different components without reloading the page.
- **Framer Motion**: Provides smooth transitions for animations between web pages.

### Technology Stack:

- **ReactJS**: A JavaScript library for building user interfaces.
- **Vite**: A modern frontend build tool that significantly improves the development experience.
- **JavaScript**: The scripting language used to create dynamic content on the web.

## Admin

The admin interface is secured with a login system to ensure that only authorized users can access the admin functionalities.

### Features:

- **Secure Login**: Protects the admin area with authentication.
- **React Router DOM**: Facilitates navigation within the admin interface.
- **React Icons**: Enhances the UI with icons for a better user experience.

## Server

The server handles all the backend processes, built on ExpressJS with various other middleware and libraries to support functionality.

### Features:

- **ExpressJS**: A web application framework for Node.js, designed for building web applications and APIs.
- **Dotenv**: Loads environment variables from a `.env` file into `process.env`.
- **Express-session**: Manages sessions for user interactions.
- **Mongoose**: Provides a straight-forward, schema-based solution to model application data.
- **Multer**: A middleware for handling `multipart/form-data`, which is primarily used for uploading files.
- **Nodemon**: A utility that monitors for any changes in your source and automatically restarts your server.
